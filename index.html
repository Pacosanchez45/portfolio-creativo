Caso UI/UX · salud privada

Clínica Dental LB

Rediseño conceptual de una web de clínica dental para mejorar claridad, confianza y solicitud de cita desde móvil. El caso muestra diagnóstico, jerarquía, CTA y propuesta visual.

  • Problema: la web informa, pero no guía con fuerza hacia la cita.
  • Solución: primera pantalla más clara, contacto visible y servicios ordenados.
  • Aprendizaje: vender una mejora requiere explicar impacto, no solo estética.
